About This Demo

Thinking in Data — This interactive map demonstrates how tabular data (CSV, JSON) can be converted to geographic formats (GeoJSON) for web-based visualization.

Try this: Click on markers to view inscription details, use the search box to filter locations, and explore how data is structured across multiple formats.
